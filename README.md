# EvoStream

EvoStream is a static multi-page website that explores the evolution of entertainment across different eras, from early live performance to gaming, digital media, music technology, and the future of immersive experiences.

## Project Link

GitHub Repository:
`https://github.com/hassanawan7890/evostream`

Live Demo:
`https://evostream.vercel.app`

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

- `index.html` is the main landing page
- `style.css` contains the shared styling across the site
- `feedback.html` is the feedback page
- `feedback-widget.js` powers the interactive feedback experience
- `images/` stores the visual assets used throughout the site

## Feedback System

The feedback page uses `feedback-widget.js` to:

- handle star ratings
- track selected topics
- save a temporary draft in local storage
- open a pre-filled email draft for sending feedback

Current feedback destination:
`hassanawan789@outlook.com`

## Deployment

This project is deployed on Vercel as a static site.

Recommended Vercel flow:

1. Open Vercel
2. Create a new project
3. Import the GitHub repository `evostream`
4. Keep the framework preset as `Other`
5. Deploy

## Local Preview

Because this is a plain static website, you can open `index.html` directly in the browser or run it with any simple local server.
